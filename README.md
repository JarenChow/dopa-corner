# dopa-corner
github corner(canvas)

# install

1. `<script src="https://cdn.jsdelivr.net/npm/dopa-corner"></script>`
2. `npm install dopa-corner --save`

# usage

```html
<a id="corner" href="https://github.com/JarenChow" style="position: fixed;top: 0;right: 0;"></a>

<script>
  createCorner('#corner');
</script>
```
